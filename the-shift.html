Node.js
Node.js היא סביבה שמאפשרת למחשב שלכם להריץ JavaScript — הקרקע שעליה כל השאר (Next.js, npm, Tailwind) עומד.

Tailwind
Tailwind היא ספרייה ענקית של classes מוכנים לעיצוב. במקום לכתוב CSS מאפס, כותבים שמות קצרים כמו p-4 (padding) או bg-black — והעיצוב פשוט עובד.
npm install tailwindcss
